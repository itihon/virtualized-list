## Features

#### Built for content that refuses to stay one size

<CardGrid stagger>
	<Card title="Dynamic heights" icon="pencil">
		Items render naturally, then the layout adapts as their measured size changes.
	</Card>
	<Card title="Responsive grids" icon="document">
    A list can become a grid as the container grows, without manual row math.
	</Card>
	<Card title="Framework adapters" icon="add-document">
		Use the core engine directly or reach for React, Vue, and Angular packages.
	</Card>
	<Card title="Window scrolling" icon="setting">
    Virtualize inside a local scroller or coordinate with the document viewport.
	</Card>
</CardGrid>
